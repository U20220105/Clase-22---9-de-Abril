# ejercicios de polimorfismo en typescript

## 📌 descripción

este repositorio contiene dos ejercicios básicos de programación orientada a objetos utilizando el concepto de polimorfismo en typescript.

---

## 🧩 ejercicios incluidos

### 1. sistema de empleados

se crea una clase abstracta `Empleado` con el método `trabajar()`.
las clases `Programador` y `Diseñador` implementan este método de forma diferente.

### 2. sistema de electrodomésticos

se define una clase abstracta `Electrodomestico` con el método `encender()`.
las clases `Televisor` y `Lavadora` implementan su propio comportamiento.

---

## ⚙️ cómo ejecutar

1. compilar los archivos:

```bash
tsc "ejercicio 1 sistema de empleados.ts"
tsc "ejercicio 2 sistema de electrodomésticos.ts"
```

2. ejecutar:

```bash
node "ejercicio 1 sistema de empleados.js"
node "ejercicio 2 sistema de electrodomésticos.js"
```
